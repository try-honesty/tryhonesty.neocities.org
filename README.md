# tryhonesty.neocities.org
This is my personal website. It's statically generated by [Eleventy](https://11ty.dev) and [deploys](https://tryhonesty.neocities.org) (via [GitHub Actions](https://github.com/features/actions)) to [Neocities](https://neocities.org).

Local debugging can be done by first pulling in the required packages by running `npm install` in the root directory of where you've `git`-pulled or extracted the repository (where [`package.json`](https://github.com/try-honesty/tryhonesty.neocities.org/blob/main/package.json) is located,) and then running `npm start` to have Eleventy spin up a hot-reloading web server [on `localhost` at port `8080`](http://localhost:8080) (by default.)
